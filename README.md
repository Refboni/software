# 🚗 Projeto: Sistema de Estacionamento
**Disciplina:** Engenharia de Software

## 📖 Sobre o Repositório
Este repositório foi criado para o armazenamento e entrega dos artefatos solicitados na disciplina de Engenharia de Software. O projeto consiste no estudo, documentação e modelagem dos processos de um estacionamento real.

## 📂 Conteúdo da Entrega
Todos os arquivos do projeto estão localizados dentro da pasta `ESTACIONAMENTO`:

* **TAP: ** Termo de Abertura de Projeto, contendo a definição inicial, escopo e objetivos do trabalho.
* **Observação de Rotina: ** Documento detalhando a atividade de observação em campo para o levantamento de requisitos e entendimento do negócio.
* **Diagrama BPMN: ** Modelagem do processo atual. Ilustra o fluxo de funcionamento e as atividades de como o estacionamento opera hoje, antes de qualquer proposta de automação ou novo sistema.
